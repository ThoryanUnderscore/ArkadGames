ARKAD GAMES

Création d'une plateforme de jeux d'arcade en Linux Bash pour un projet scolaire
La plateforme est gérée par le script menu.sh et peut appeler tous les jeux disponibles dessus, chacun dans son script respectif. 

Il y a 7 jeux différents :
SNAKE
MASTERMIND
PUISSANCE 4
PENDU
LABYRINTHE
MEMORY
SIMON

Chacun est totalement indépendant du reste évitant les erreurs et les conflits de fichiers/données.

Le code est documenté sur ses fonctionnalités.
